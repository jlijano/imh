# Iloilo Mission Hospital Website

Static responsive website for Iloilo Mission Hospital, also known as IMH or CPU-Iloilo Mission Hospital.

## Pages

- Home
- About IMH
- Services
- Doctors & Departments
- Patient Guide
- Training & Education
- News & Updates
- Contact

## Editing Notes

This site is intentionally built with plain HTML, CSS, and JavaScript so maintainers can edit content without a build step.

Public pages should use verified hospital facts only. Do not publish doctor names, schedules, HMO lists, visiting hours, personal contact details, clinical claims, or training requirements until authorized hospital staff confirm the exact wording.

When information is not confirmed, use patient-safe wording that directs visitors to contact the hospital directly for current availability.

## Safety

The site keeps emergency language visible and includes a general medical information notice. Website content must not be used for emergency triage, diagnosis, or treatment advice.
