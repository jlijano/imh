# Image assets

Place optimized public hospital images here for the homepage gallery.

Recommended files:

- `imh-courtyard.webp` - primary courtyard image from the uploaded file `01-imh-2-.webp`.
- `facebook-hospital-building.webp` - approved/exported building or campus image from https://www.facebook.com/iloilomissionhospital.
- `facebook-care-team.webp` - approved/exported staff, event, or patient-care environment image from the Facebook page.
- `facebook-facilities.webp` - approved/exported department, facility, or service environment image from the Facebook page.

Use WebP where possible, keep files reasonably compressed, and prefer dimensions around 1200px wide for large images and 900px wide for gallery cards.
